### Сетап проекта
```
1. git clone https://github.com/aliadinov/yii2-apples
2. composer install
3. php init --env=Development
4. Создать БД apples и прописать ее в common/config/main-local.php
5. php yii migrate
```

### Комментарии по коду:
```
1. Дату появления выставлять случайной показалось странным. Сделал стандартным способом через TimestampBehavior. 
2. После лежания 5 часов - портится. Сделал в методе AfterFind класса Apple, для данной задачи этого достаточно, для более сложных проектов можно добавить крон задачу, а также обновлять статусы яблок непосредственно перед выполнение задач  связанных с яблоками.
3. Все формы умеют отправлять как обычным постом с перегрузкой страницы: так и аяксом. Избыточно, но вот так сделал. 
```
